# 日本語版アーカイブ（_ja-archive）

このフォルダは、サイトを英語化する前の**日本語版HTMLのスナップショット**です（2026-07-04 時点）。

## 目的
- 英語化にあたり、元の日本語テキストをデータとして保全するため。
- いつでも内容を参照・復元できるようにするため。

## 公開されないことの保証（三重）
1. **Jekyll によるビルド除外**: このリポジトリには `.nojekyll` が無いため GitHub Pages は Jekyll でビルドします。Jekyll は `_` で始まるフォルダを出力に含めないため、`/_ja-archive/` は公開URLとして配信されません。
2. **robots.txt**: `Disallow: /_ja-archive/` を追加済み。
3. **sitemap 非掲載**: sitemap.xml には一切含めていません。

そのため、英語版サイトと競合する日本語ページは生成されず、AdSense 審査上の重複コンテンツにもなりません。

## 復元方法
特定ページの日本語版に戻したい場合は、対応するファイルを一つ上の階層へコピーします。例:

```
cp _ja-archive/about.html about.html
```

なお、この時点の完全な日本語版は git タグ `ja-full-2026-07-04` にも保存されています。
