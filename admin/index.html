接続設定（GitHub）

トークンとリポジトリの設定を開く

このリポジトリ専用の fine-grained トークンを推奨。トークンはあなたのブラウザ内（localStorage）にのみ保存され、外部には送られません（送り先は GitHub API のみ）。
