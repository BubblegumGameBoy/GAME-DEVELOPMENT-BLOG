The typing games are Japanese typing practice: you type hiragana words using romaji (roman letters). The game interfaces are in Japanese — a fun challenge for Japanese learners and typists alike.

About This Site

Bubblegum Game is a collection of free browser games handmade by "Bubblegum", a solo indie developer in Japan.

I'm a lifelong gamer who loves making games as much as playing them. Everything here is a lightweight HTML game you can enjoy the moment the page loads — if you like what you find, bookmark the site! I aim to update it about twice a month.
